This folder documents my complete Machine Learning journey — from foundational concepts in ML I to advanced techniques in ML II, alongside personal self-learning experiments I run outside of coursework.

My Learning Philosophy
Machine Learning is not about memorizing algorithms.
It is about understanding why a model behaves the way it does — and knowing when to use it, when not to, and how to explain it to a client or stakeholder.

Learning in public. Building in depth.
