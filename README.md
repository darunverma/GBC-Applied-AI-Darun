# Extrasss

## 📌 Overview
This folder is the dump zone.

Anything that does not fit neatly elsewhere lives here —
random experiments, half-finished ideas, interesting findings,
articles I am working through, code I am just trying out,
and thoughts that have not become projects yet.

No structure. No deadlines. No pressure.
Just curiosity in its rawest form.

## My Learning Philosophy
Not everything needs a purpose right away.
Some of the best ideas start as a mess in a corner folder.

Learning in public. Dumping with intent.
