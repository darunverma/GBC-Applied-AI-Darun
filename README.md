# Foundations of Data Management

## 📌 Overview
This folder documents my coursework from AASD4002 — Foundations of Data Management.

Data is only as useful as the system that stores, organizes,
and retrieves it. This course builds the foundation —
databases, data models, querying, and the discipline of
managing data like it actually matters.

It is a live, honest record. You will see queries that fail
before they work, schemas that get redesigned, and concepts
revisited more than once. That is intentional.

## My Learning Philosophy
Bad data management does not just slow down a project —
it corrupts the decisions made from it.

I am learning to treat data with structure and respect
from the very first line.

Learning in public. Building in depth.
